\#  TP\_Persistance



\## Description

Ce projet est une application web de vote électronique réalisée avec Flask et MySQL.



Il permet aux utilisateurs de :

\- s'inscrire

\- voter pour un candidat

\- consulter les résultats

\## Fonctionnalités

\- Inscription des électeurs

\- Vote sécurisé

\- Empêchement du double vote

\- Affichage des résultats



\## Technologies utilisées

\- Python (Flask)

\- SQLAlchemy (ORM)

\- MySQL

\- HTML / CSS



\##  Sécurité

\- Mot de passe hashé

\- Un seul vote par utilisateur (`has\_voted`)
